Technologies we use

Windows

Windows Server 2019 for a server with Active Directory and Group Policies. Windows 10 for clients.

Debian

Debian Stable for the webserver. Our preferred webserver application is Apache 2. We use ProFTPD for FTP server.

MikroTik

We use MikroTik RouterOS 6.48.2 for running a virtual router in our virtual network. It's easy to set up.
